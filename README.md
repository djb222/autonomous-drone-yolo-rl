# Autonomous Drone Goal Detection — YOLO + PyBullet + Reinforcement Learning

## Overview

This project involved developing and testing an autonomous drone simulation in Python. The simulated drone navigates toward goal objects while integrating computer vision through a YOLO-based detection pipeline.

The project combines simulation, perception, and autonomous control concepts, making it relevant to robotics, space systems, autonomous navigation, and software-based engineering applications.

## My Contribution

This was a group university project. My main contribution focused on the perception and simulation integration side of the system.

I contributed to:

- Integrating YOLO object detection into the drone simulation workflow.
- Configuring camera/image capture from the simulated drone environment.
- Testing object detection behaviour across different positions, camera settings, and confidence thresholds.
- Connecting perception outputs to the environment information used by the navigation/control system.
- Debugging simulation issues related to model loading, image processing, and environment behaviour.
- Supporting the final system demonstration and project documentation.
